---

## 🤝 How to contribute
If you know a great open-source program that is not on the list:
1. Fork this repository.
2. Add the program to the appropriate category using the template: `* [**Name**](Official Site Link) — Short description. ([Review](Review Link))`
3. Submit a Pull Request.

If the software is truly FOSS, we will add it to the catalog and later publish a detailed review on [getfoss.org](https://www.getfoss.org/)!
